VIAF bookmarklet installation page

This page offers a bookmarklet which can be installed as bookmark in your browser for detection of VIAF numbers on web pages when clicking then on it.

General warning:
Adding of bookmarklets from other sources can be dangerous if the source and server of Javascript cannot be trusted.
Bookmarklets can reload further scripts which can be malicious.
Bookmarklets and further scripts can possibly access your local information.

The scripts on this page are safe. T. Gries

  • VIAF ← bookmarklet
    • ← left-click, keep it and drag and drop that to your Firefox or Chrome bookmark bar ("Lesezeichen-Symbolleiste")
    ← Internet Explorer users right-click and install ("Add as favorite") in the favorite bar ("Favorite bar")
    ← (de) Internet Explorer rechts klicken > "Zu Favoriten hinzufügen..." > "Möchten Sie diesen Vorgang fortsetzen?" (Ja) > "Favoritenleiste" (!) auswählen
